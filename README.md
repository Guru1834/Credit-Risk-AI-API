# 📊 Credit Risk AI API

A production-ready Machine Learning API that predicts a normalized credit score (0–100) based on an individual's financial behavior and risk profile.

## 🚀 Features
- Financial feature engineering (DTI, Expense Ratio, Savings Ratio, Disposable Ratio)
- Random Forest regression model
- Risk classification (Low / Moderate / High)
- Smart financial recommendations
- Dockerized FastAPI deployment
- Hosted on Hugging Face Spaces

## 🛠 Tech Stack
Python • Pandas • Scikit-learn • FastAPI • Docker • Hugging Face

## 🌍 Live API

Deployed on Hugging Face Spaces:

🔗https://gurujod-csp.hf.space/predict

Returns credit score, risk level, financial metrics, and recommendations in structured JSON format.
